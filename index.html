Membuat Web Toko Ikan Hias Sederhana

Saya membuat web toko ikan hias sederhana menggunakan HTML, CSS, dan JavaScript. Web ini memiliki fitur untuk menampilkan produk ikan hias, serta informasi tentang harga dan deskripsi produk.

Tantangan : Saya menghadapi tantangan dalam membuat desain web yang responsif dan mudah digunakan. Saya juga harus memastikan bahwa web dapat diakses dengan baik pada berbagai perangkat.

Membuat Jaringan Mikrotik

Saya membuat jaringan Mikrotik untuk menghubungkan beberapa perangkat dalam satu jaringan. Jaringan ini memiliki fitur untuk mengatur akses internet dan membatasi akses ke situs tertentu.

Tantangan : Saya menghadapi tantangan dalam mengkonfigurasi jaringan Mikrotik agar dapat berfungsi dengan baik dan aman. Saya juga harus memastikan bahwa jaringan dapat diakses dengan baik oleh semua perangkat.

Membuat Desain Poster Sekolah

Saya membuat desain poster sekolah menggunakan aplikasi Canva. Poster ini memiliki desain yang menarik dan informatif, serta dapat digunakan untuk mempromosikan sekolah.

Tantangan : Saya menghadapi tantangan dalam membuat desain poster yang menarik dan efektif. Saya juga harus memastikan bahwa poster dapat dibaca dengan baik dan memiliki ukuran yang sesuai.

Membuat Web Music Player

Deskripsi : Saya membuat web music player yang memungkinkan pengguna untuk memutar musik secara online. Web ini memiliki fitur untuk memutar musik, menampilkan daftar lagu, dan mengatur volume.

Teknologi yang digunakan : HTML, CSS, Framework bootstrap, dan JavaScript.

Tantangan : Saya menghadapi tantangan dalam membuat pemutar musik yang dapat berfungsi dengan baik pada berbagai browser dan perangkat. Saya juga harus memastikan bahwa web dapat memuat musik dengan cepat dan lancar.
